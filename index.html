About Me

Hey, I'm Tufelizamigo. I run The Pure Vanilla SMP and lead tufelizteam. Most of my time goes into the server — setting up rank hierarchies, the economy, custom Skript tooling, and the bits of code that hold it all together.

Outside the server I'm a web tinkerer. I build my own projects from scratch, like PureVanilla Connection, a bridge that pipes our Discord chat straight into the game.

When I'm not building, I'm playing. I'm into Beat Saber on my standalone Quest, and I've spent a lot of hours in Geometry Dash — both playing levels and making my own. I like figuring out how things work and then making them better.
